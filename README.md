![Status](https://img.shields.io/badge/Project-Active-success)

A web-based **E-Learning Platform** that allows students to enroll in **free and paid courses**, instructors to apply for teaching roles, and administrators to manage the platform through a centralized dashboard.

The system is designed to provide a **simple, organized, and scalable online learning environment**.

---

## 🚀 Project Overview

The platform provides an integrated system where users can explore courses, enroll in learning programs, and interact with administrators.

The goal of the platform is to improve accessibility to online education through an organized and user-friendly system.

---

## 🎯 Core Features

### 👨‍🎓 Student Features

- User registration and authentication
- Browse available courses
- Enroll in free or paid courses
- Upload payment proof for paid courses
- Track enrollment status
- Apply to become an instructor
- Contact administrator

---

### 👨‍🏫 Instructor Features

- Submit instructor application
- Track application approval status
- Manage courses after approval

---

### 🛠 Admin Features

- Add, update, and delete courses
- Manage course categories
- Approve or reject enrollments
- Verify payment proofs
- Approve or reject instructor applications
- Manage users and course content
- Send email notifications
